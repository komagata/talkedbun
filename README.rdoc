= talkedbun

Talkedbun "喋饅頭" is japanese text-to-speech server.

== Installation

  % sudo gem install komagata-talkedbun -s http://gems.github.com
  % talkedbun
  % open http://localhost:4567/

required SayKana, ffmpeg command.
now, MAC OS X only...
